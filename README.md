# 📌 Overview

SmartCRM is a modern Customer Relationship Management (CRM) dashboard developed completely using Python.

The system helps businesses:
- Manage customer records
- Track business analytics
- Store customer information
- Analyze company statistics
- Handle CRM operations efficiently

This project is fully developed using:
- Python
- Streamlit
- SQLite
- Pandas
